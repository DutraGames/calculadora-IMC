# Calculadora IMC

A realização de um simples aplicativo em React Native, mas que a função Calcular no (App.js), pode ser utilizada para a realização em qualquer outro código que deseje usar o cálculo de IMC.

## 👨‍💻Technology:

 * Expo
 * React
 * React Native
